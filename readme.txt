== Description ==

It's a complete social networking websites content share plugin. Sprybirds.com, Facebook.com, Twitter.com and other.


== Installation ==

1. Upload `all-sociable` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
